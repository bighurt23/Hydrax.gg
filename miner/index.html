Run on Linux or HiveOS

Grab the build for your OS above, then from a terminal:

Linux
wget https://github.com/bighurt23/hydrax-miner/releases/download/v3.4.5/HydraX-miner-v3.4.5-linux64.tar.gz
tar -xzf HydraX-miner-v3.4.5-linux64.tar.gz
cd HydraX-miner-v3.4.5-linux64
nano mine-pearl-mdl-zeph.sh   # put your wallet in
bash mine-pearl-mdl-zeph.sh
HiveOS
wget https://github.com/bighurt23/hydrax-miner/releases/download/v3.4.5/HydraX-miner-v3.4.5-hiveos-cuda12.tar.gz
tar -xzf HydraX-miner-v3.4.5-hiveos-cuda12.tar.gz
cd HydraX-miner-v3.4.5-hiveos-cuda12
nano mine-pearl-mdl-zeph.sh   # put your wallet in
bash mine-pearl-mdl-zeph.sh

Prefer another coin? Any mine-*.sh launcher works the same way. Solo Yerbas: bash mine-yerbas.sh.
